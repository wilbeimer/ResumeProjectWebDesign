Certifications

AWS Cloud Practitioner

During my high school dual enrollment Computer Science class, I pursued the AWS Certified Cloud Practitioner certification. As part of this course, I studied core cloud concepts, services, and best practices, dedicating time in class and independently to mastering topics such as cloud computing fundamentals, security, and basic AWS architecture. This certification challenged me to apply what I was learning in real-world scenarios, and achieving it demonstrated my commitment to understanding cloud technologies at an early stage in my academic journey.

Azure Cloud Fundamentals

In addition to AWS, I earned the Microsoft Certified Azure Fundamentals certification while continuing my dual enrollment studies. Through guided lessons and hands-on exercises, I learned to navigate Microsoft cloud solutions, explore core platform features, and implement best practices for cloud applications. Preparing for this certification helped me strengthen my technical skills and reinforced my ability to learn and succeed in professional-level technology environments even as a high school student.
